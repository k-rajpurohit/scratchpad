# scratchpad

Personal scratchpad for code snippets, notes, and small experiments.

## What lives here

- One-off scripts I want to keep but don't deserve their own repo.
- Reproducers for issues / questions.
- Tiny utilities.

## Structure

```
.
├── snippets/   # Standalone scripts (Python, Bash, JS)
├── notes/      # Markdown notes
└── experiments/ # Throwaway prototypes
```
